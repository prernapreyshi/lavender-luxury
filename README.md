#  Lavender Luxury — Full-Stack E-Commerce Platform

Lavender Luxury is a modern full-stack e-commerce application designed to deliver a smooth and elegant shopping experience. It combines a fast, responsive frontend with a secure and scalable backend.

The platform allows users to browse products, manage their cart, and place orders seamlessly, while also supporting authentication and product management.

---

##  Key Features

-  Dynamic product browsing with category filtering  
-  Secure user authentication using JWT  
-  Real-time cart management  
-  Order creation and tracking  
-  Fast and responsive UI across all devices  
-  Clean and modular full-stack architecture  

---

##  Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- shadcn/ui

### Backend
- Node.js + Express.js  
- Supabase (Database & Auth support)  
- JWT Authentication  

---


##  Overview

Lavender Luxury is built with a focus on performance, scalability, and user experience.  
It demonstrates how a modern e-commerce platform can be structured using a clean separation between frontend and backend, while maintaining smooth communication through REST APIs.
