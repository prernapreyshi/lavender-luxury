#  Lavender Luxury – E-Commerce Website

Lavender Luxury is a modern, responsive e-commerce web application focused on delivering a smooth and intuitive shopping experience. The project demonstrates strong frontend fundamentals, reusable component architecture, and real-world e-commerce features.

 **Live Demo:** https://lavender-luxury.vercel.app/

---

##  Features
- Responsive and modern UI
- Product listing with filtering
- Shopping cart (add/remove/update items)
- Reusable React components
- Optimized state management
- Clean styling using Tailwind CSS

---

## Tech Stack
**Languages & Tools:**  
HTML, CSS, JavaScript, React.js, Tailwind CSS
